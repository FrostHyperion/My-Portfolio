HTML AND CSS

HTML or “HyperText Markup Language”. It’s a programming language that helps us to display the text, images, and links on the web browser. Apart from that, you may be wondering why we need CSS or “Cascading Style Sheets”. CSS helps users to add styles to make the website look more responsive. For Example: suppose you are making a car from scratch. Firstly, you build the body of the car without paint, wheels, and function. This is like Html. Secondly, we customize the car by painting the car, installing wheels, mirrors, etc. That is the CSS part.

More about CSS

Now, Html basics are done. Moving into CSS or Cascading Style Sheets. There are some CSS rules before defining CSS. First, we have a CSS selector which connects to the elements of HTML to assign certain CSS properties to the selected HTML elements. A CSS rule is called when one or more selectors are applied to one or more selected Html elements. CSS values are the property inside the CSS declaration block. Example: blue, percentage, length, width, etc.
